Projects

Here are some of my projects as a student:

  • Diploma Thesis
    I did my diploma thesis during my internship at INRIA, under the supervision of Cătălin Hriţcu. My thesis involved the design and the implementation of a verification mechanism aimed at proving the correctness of random test data generators and testing code, with respect to high level declarative specifications. (Internship Report, Thesis, Slides, Code)

  • Compiler for Llama
    During a compilers course at NTUA, along with 2 other students, I developed a compiler for a functional OCaml-like language. The implementation featured type inference, high ordered functions, user defined data types and patter matching. The compiler performs various optimizations, such as copy propagation, common subexpression elimination, dead code elimination, tail call elimination and peephole optimizations. (Code)
  • Applications of Logic in CS Project
    As a project for the course Applications of logic in Computer Science, I implemented in Haskell, along with an other student, an interpreter for a lambda calculus variant, featuring Hindley–Milner type inference. (Code)
  • Advanced Topics in Database Systems Project
    A bibliographic report about security and cryptography in database systems. (Report)
  • Cryptography Project
    A library implementing basic operations on elliptic curves over prime fields, as well as Elliptic Curve digital signature scheme and Diffie-Hellman key exchange algorithm. Developed in Ocaml in a team of 2 students. (Code)
